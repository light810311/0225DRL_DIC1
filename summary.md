# Light | Personal Workspace

## Project Overview
This project is a modern, visually appealing personal workspace webpage featuring a dynamic clock and date display. It is built with a focus on UI/UX, incorporating glassmorphism design principles, smooth animations, and a fully responsive layout.

## Key Features
*   **Real-time Clock & Date**: Displays the current time and date dynamically.
*   **Glassmorphism UI**: Utilizes modern "frosted glass" layered effects.
*   **Animated Elements**: Includes smooth fade-in and structural animations for an enhanced user experience.
*   **Responsive Design**: Adapts beautifully to different screen sizes and orientations.

## Technologies Used
*   **HTML5**: For structure and semantics.
*   **CSS3**: For styling, Flexbox, CSS Grid, Glassmorphism techniques, and Keyframe animations.
*   **Vanilla JavaScript**: For the logic behind real-time clock updating and DOM manipulation.
